## 重み付きグラフ

## 木

- 木とは: サイクルのない無向グラフ 

```mermaid
graph TD
    0---1 
    1---4
    1---5
    0---2
    2---6
    2---7
    0---3
    3---8
    3---9
```

## heapの定義

- 頂点$v$の親を$p$としたとき、key[p] $\ge$ key[v]
- 木の高さが$h$のとき木の深さが$h - 1$以下の部分については完全二分木を形成している
- 木の高さ$h$としたとき、木の深さ$h$の部分については頂点が左詰められている。
